# 🌟 LinguaSub AI

**LinguaSub AI** is a premium, AI-powered Chrome Extension designed to automatically generate, translate, and overlay highly realistic subtitles for any online video.

## ✨ Key Features

-   **Seamless AI generation**: Powered by Google's **Gemini 2.5 Flash**.
-   **Premium Interface**: A modern, glassmorphic UI that focuses purely on your language selection.
-   **Multi-Language Support**: Supports Hindi, Telugu, Tamil, Malayalam, English, Spanish, and more.
-   **Robust Overlay**: Works on modern video platforms (like YouTube), including support for **Fullscreen** mode.
-   **Exportable Subtitles**: Download your generated subtitles as `.srt` or `.vtt` files anytime.

## 🛠️ Installation (Developer Setup)

1.  **Clone / Download** this repository.
2.  **API Setup**:
    *   Find the `background/config.example.js` file.
    *   **Copy** it and rename the copy to `background/config.js`.
    *   Open `background/config.js` and paste your Gemini API key from [AI Studio](https://aistudio.google.com/app/apikey).
3.  Open Chrome and navigate to `chrome://extensions`.
4.  Enable **Developer Mode** in the top right corner.
5.  Click **Load unpacked** and select this project folder.
6.  **Pin** LinguaSub AI to your browser toolbar for quick access!

## 🎬 How to Use

1.  Navigate to any video (e.g., YouTube).
2.  Open the **LinguaSub AI** popup.
3.  Select your desired **Subtitle Language**.
4.  Click **Generate Subtitles**.
5.  **Enjoy!** The subtitles will automatically overlay the video as it plays.
